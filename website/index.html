Why Gemini.

1M Token Context

Hold your entire project, documentation, and history in memory at once.

Google Search

Live research with cited sources. Not based on stale training data, but real-time reality.

Multimodal Input

Feed the engine screenshots, complex PDFs, diagrams, and images as core context.

Context Caching

90% cost reduction on repeated complex queries by persistent state storage.

Not another coding CLI

It is an information tool first. While code can run behind the scenes to process data, the interface is built for human thinkers and writers, not just devops.

Not a chatbot

It doesn't just reply; it produces artifacts. Final documents, presentations, research graphs, and executable scripts are the primary goal.

Not browser-bound

Sessions persist locally on your hardware. Folders stay where you want them. No browser tab loss, no cloud-only lock-in.

Not model-agnostic

By being Gemini-native, gKestral exploits the model's best features -- like massive context and search grounding -- to their absolute limit.
