<task id="T1" wave="3" depends_on="T2,T3" complexity="complex">
  <files>internal/google/auth/oauth.go, internal/google/auth/oauth_test.go</files>
  <action>
    Build Google OAuth 2.0 desktop app flow with PKCE:

    - OAuthConfig struct: ClientID, ClientSecret, RedirectURL (localhost loopback),
      Scopes []string, AuthURL, TokenURL
    - NewOAuthClient(opts ...OAuthOption) *OAuthClient with functional options:
      * WithClientID(id string)
      * WithClientSecret(secret string)
      * WithScopes(scopes ...string)
      * WithCallbackPort(port int) -- default 8085
      * WithTokenStore(store TokenStore)
    - DesktopFlow(ctx context.Context) (*oauth2.Token, error):
      * Generate PKCE code_verifier (43-128 chars, [A-Za-z0-9-._~])
      * Compute code_challenge = base64url(SHA256(code_verifier))
      * Build auth URL with response_type=code, code_challenge, code_challenge_method=S256,
        include_granted_scopes=true (CRITICAL for progressive permissioning --
        without this, new tokens drop previously granted scopes)
      * Start local HTTP server on localhost:PORT for redirect callback
      * Generate cryptographic state parameter (32 bytes, base64url)
      * Open system browser to auth URL (use exec.Command for cross-platform)
      * Wait for callback with auth code (with timeout: 5 minutes)
      * Validate state parameter in callback matches generated value (CSRF protection)
      * Handle error callbacks: access_denied, server_error, temporarily_unavailable
      * Exchange code for tokens at token endpoint (include code_verifier)
      * Shut down local server (on ALL exit paths: success, error, timeout, cancel)
      * Store tokens via TokenStore
      * Return token
    - Edge cases (must be tested):
      * Callback port already in use: try next port or return ErrPortInUse
      * Duplicate callbacks: ignore second callback, close connection
      * User denies consent (error=access_denied): return ErrConsentDenied
      * Context cancelled during wait: clean shutdown of HTTP server
      * State mismatch: reject callback, return ErrStateMismatch
    - AuthenticatedClient(ctx) (*http.Client, error) -- returns http.Client
      with automatic token injection and refresh
    - Standard endpoints:
      * AuthURL: https://accounts.google.com/o/oauth2/v2/auth
      * TokenURL: https://oauth2.googleapis.com/token

    Use golang.org/x/oauth2 as the token management foundation.
    PKCE must be implemented manually (oauth2 package doesn't enforce S256).

    Test with mock HTTP server: token exchange, PKCE verification,
    browser open mock, callback handling, timeout behaviour.
  </action>
  <verify>go test ./internal/google/auth/ -run TestOAuth passes</verify>
  <done>Desktop OAuth flow with PKCE completes token exchange via mock server</done>
