## Trademark Classification

For a software product like gKestral, the relevant Nice Classification classes are:

### Class 9 (Primary -- Software)
- Computer software
- Downloadable software applications
- Desktop software for information management and AI-assisted productivity

### Class 42 (Secondary -- SaaS/Services, if applicable later)
- Software as a service (SaaS)
- Cloud computing services
- Computer software design and development

## Filing Options

### Option 1: Saudi Arabia (SAIP)
- Saudi Authority for Intellectual Property
- Filing: online via saip.gov.sa
- Cost: ~2,000-3,000 SAR per class
- Timeline: 6-12 months
- Protection: Saudi Arabia only

### Option 2: UAE (if Aarksee has UAE entity)
- Ministry of Economy
- Cost: ~6,000-10,000 AED per class
- Timeline: 6-12 months

### Option 3: International (Madrid Protocol via WIPO)
- File through home country office, designate multiple countries
- Cost: ~653 CHF base + per-country fees
- Covers 130+ countries
- Best for global product ambitions

### Option 4: United States (USPTO)
- Direct filing if planning US market
- Cost: $250-350 per class (TEAS Plus)
- Timeline: 8-12 months
- Strongest for tech products

### Option 5: European Union (EUIPO)
- Single filing covers all 27 EU member states
- Cost: ~850 EUR for first class
- Timeline: ~4-6 months
